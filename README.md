# Code

### to follow workshop:
```
$ git checkout dev
$ node server.js
```

### to dev latest
```
$ git checkout master
$ npm start
```

### Create a .env file on root

```
NODE_ENV=development
PORT=4000
MONGO_DB_CONNECTION_STRING=YOUR_CONNECTION_STRING_HERE
# API keys go here
```
